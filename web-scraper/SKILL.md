# Web Scraper Skill

Use this skill when you need the complete readable text and structure from a given website URL. It uses standard Python libraries to fetch the HTML and strip away noise, returning plain text so the Language Model can understand real-time page content.

## Features

- URL content fetching.
- Strips javascript and styles.

## Usage

```bash
python c:/Users/ehrha/.copilot/skills/web-scraper/fetch.py "https://example.com"
```
